Getting started

Three steps.
That is it.

01
Add to your server
Click the invite link and authorize Panda Bot with Administrator permissions. All 64 slash commands will register automatically within a few seconds.
discord.com/oauth2/authorize
02
Configure basic settings
Use the administration commands to set up your server. Set a welcome channel, log channel, and mod role. Everything else works with no configuration.
/setwelcome /setlogchannel /setmodrole
03
Start using commands
Type / in any channel to see all available commands. The leveling system, economy, and anti-spam features are all active immediately with no extra setup.
/rank /ticket /gstart /slots
